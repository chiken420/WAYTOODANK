Random Access Memory aka RAM

Random Access Memory, also known as RAM, is a type of hardware inside tons of different devices like your phone and computer. It is used as flash-storage, which is pretty much temporary storage, to store some working part of the operating system temporarily, and it will do the same with data from applications that are currently active. Functionality of RAM

Because of its volatility, RAM is unable to premanent information. Its short-term memory is focused on one thing, the work that is currently being processed by the computer. It can only keep certain information for a certain amount of time. If the RAM is to get dull, the CPU, something of which we will get into later, will go to the hard disk, something which we will also get into later, repeatedly.
